## ignite ps

List running VMs

### Synopsis


List all running VMs. By specifying the all flag (-a, --all),
also list VMs that are not currently running.


```
ignite ps [flags]
```

### Options

```
  -a, --all    Show all VMs, not just running ones
  -h, --help   help for ps
```
